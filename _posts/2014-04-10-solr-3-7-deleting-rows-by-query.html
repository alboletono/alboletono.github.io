In JSON you can delete directly in the index rows that fit a query.

E.g.: in the following you can delete all results that have an url containing keyword total. Then it is directly committed.

[code language="bash"]

curl http://localhost:8983/solr/update/json -H 'Content-type:application/json' -d '

{
"delete": { "query":"url:*", "commitWithin":500   }
}'

[/code]
