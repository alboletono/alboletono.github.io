--- layout: post title: Installing D-Link AirbuPlus G+ wifi PCMCIA card on linux (xBuntu) date: 2014-09-06 12:35:39.000000000 +02:00 type: post published: true status: publish categories: - howto tags: - linux - ndiswrapper - wifi ---
  1. Download the Windows XP driver: www.dlink.com/uk/en/support/product/dwl-g650plus-54mbps-pcmcia-cardbus-802-11g-wireless-adapter?revision=deu_revb#downloads and name it drivers.zip into your home.
  2. unzip drivers.zip
  3. cd drivers/
  4. unshield ~/drivers/English_Language/*.cab (apt-get install unshield)
  5. mkdir ~/install_wifi
  6. cp ~/drivers/English_Language/DrvXP ~/install_wifi
  7. cp ~/drivers/English_Language/InfXP ~/install_wifi
  8. cd ~/install_wifi
  9. A little trick: mvÂ  GPlus_XP.sys T1130_XP.sys
  10. sudo ndiswrapper -i GPLUS.inf (apt-get install ndiswrapper ndiswrapper-dkms)
  11. Adding module: sudo ndiswrapper -m
  12. sudo modprobe -r ndiswrapper ; sudo modprobe -a ndiswrapper
  13. sudo iwconfig
  14. Then you can configure your wifi interface!
